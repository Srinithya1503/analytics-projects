# 📊 Power BI Dashboards Portfolio

Welcome to my **Power BI Dashboards Repository**, a curated collection of interactive dashboards built using real-world datasets from various sources such as **Kaggle**, **Maven Analytics**, guided projects, and **web-scraped data (e.g., IMDB)**.

This repository showcases my journey in **data analytics, business intelligence, and data storytelling** using Power BI.

---

## 🎯 Objective

The goal of this repository is to:

- Transform raw datasets into **interactive and insightful dashboards**
- Practice and implement **data modeling techniques**
- Develop strong **business understanding through data**
- Improve **data storytelling and visualization skills**
- Build a portfolio aligned with **industry Data Analyst roles**

---

## 🧠 Key Learning Focus

This repository is not just about dashboards — it reflects my learning in:

### 📌 Data Modeling
- Building relationships between tables  
- Designing **star and snowflake schemas**  
- Creating efficient data models for performance  

### ⚙️ Data Transformation
- Using **Power Query (ETL)** for:
  - Data cleaning  
  - Handling missing values  
  - Data type conversions  
  - Feature creation  

### 📊 DAX (Data Analysis Expressions)
- Calculated columns and measures  
- Time intelligence functions  
- KPI calculations  

### 📈 Data Visualization & Storytelling
- Designing intuitive dashboards  
- Choosing the right visuals  
- Creating **insight-driven reports**
